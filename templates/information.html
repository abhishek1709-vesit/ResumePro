Why this project?

I completed learning html,css and js along with python and flask. Then i wanted to make something which includes all things i learnt till now along with a little bit of AI integration using API keys

How it works?

1. The content from the entered pdf is extracted using python library PyPdf2.

2. The extracted content is then passed to the AI which is deepseek. Got an free deepseek api key from OpenRouter also used their documentations for using the api key

3. Deepseek then reads the extracted/raw content and generates analysis based on that extracted content

4. This analysis by deepseek is then shown to the user

5. For the "Chat with AI" feature the same extracted text from the pdf is send to AI along with the user questions which are took as input in the "/chat" route
